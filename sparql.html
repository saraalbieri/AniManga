Validazione tramite SPARQL Queries

Esempio di query semantica strutturata per verificare l'estrazione degli archetipi eseguiti sui casi di studio.

Query 1: Estrazione dei Personaggi associati all'archetipo "Shonen Hero"

PREFIX ex: <https://animemanga.schema.org/ontology/>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>

SELECT DISTINCT ?characterName ?archetypeLabel
WHERE {
  ?char ex:hasArchetype ex:ShonenHero .
  ?char rdfs:label ?characterName .
  ex:ShonenHero rdfs:label ?archetypeLabel .
}

Risultati simulati dall'endpoint:

?characterName?archetypeLabel
"Monkey D. Luffy"@it"Eroe Shonen"
"Uzumaki Naruto"@it"Eroe Shonen"
